Phase 0 — Prerequisites

Environment Setup & Repo Clone

Clone & Install Manual

Step-by-step Instructions
  1. Make sure you have Node.js 18+ and Git installed.
  2. Clone the repository:
    git clone https://github.com/J-Akiru5/usc-attendance.git cd usc-attendance
  3. Install dependencies:
    npm install
  4. Copy the environment file:
    cp .env.example .env
  5. Fill in the .env file with the following values (get these from the project owner):
    # Supabase SUPABASE_URL=https://your-project.supabase.co SUPABASE_ANON_KEY=your-anon-key SUPABASE_SERVICE_ROLE_KEY=your-service-role-key # Prisma (Supabase session pooler, port 5432) DATABASE_URL=postgresql://postgres.[ref]:[password]@aws-0-[region].pooler.supabase.com:5432/postgres # Seed admin password (for seed_auth.ts) SEED_ADMIN_PASSWORD=your-admin-password-here
  6. Generate Prisma client:
    npx prisma generate
  7. Start the dev server:
    npm run dev
  8. Open http://localhost:5173 — you should see the existing USC site.
💡
For AI-assisted developers: You can paste each phase's prompt into an AI coding agent (like Antigravity, Cursor, etc.) one at a time. Complete each phase and verify it works before moving to the next.

Phase 2 — Data Layer

Events System — JSON-Based Data

What Needs To Happen

Replace the hardcoded events.ts TypeScript file with a JSON-based system so events (including media — photos and videos) can be easily updated by editing a single JSON file. No database or admin UI needed.

  • Create public/data/events.json — the single source of truth
  • Each event gets: title, date (ISO), endDate, description, location, venue, status (upcoming/completed), featured flag, cover image, gallery (array of image paths), trailer video URL, and tags
  • Add a public/media/events/ folder for event images and video trailers
  • Create a composable src/composables/useEvents.ts that fetches and parses the JSON at runtime

Phase 3 — Landing Page

Hero Spotlight, Countdown & Events Carousel

🎯
This is the highest-impact phase. The landing page will get a "Featured Event" hero mode with a countdown timer, and the About section gains a horizontal events carousel. The existing animated GIF logo must be preserved.

3A: Hero Spotlight Mode

When there's a featured event (an event in events.json with featured: true and status: "upcoming"), the hero section transforms to showcase that event with:

  • Event title + countdown timer (days, hours, minutes, seconds)
  • Event cover image or trailer as the hero background
  • CTA buttons: "View Details" and "Watch Trailer" (if trailer exists)
  • The animated USC logo GIF remains visible — positioned on the right side as it currently is
  • A subtle "switch" or visual indicator showing this is an event spotlight
  • When no featured event exists, the hero falls back to the current default view

3B: Events Carousel — Under About Section

A horizontal scrolling carousel of recent/upcoming events placed directly after the "Who We Are" about preview section on the homepage. Inspired by the IRJICT website's event cards (see reference image).

  • Horizontal scroll with snap points — auto-scrolls every 5 seconds, pauses on hover
  • Each card shows: cover image, status badge (upcoming/completed), title, date, location, short description, "View Details →" link
  • If the event has a trailer, show a small "Watch →" button with a video thumbnail
  • Cards have the same navy + gold styling as the rest of the site
  • Mobile: stack vertically or scroll horizontally

Phase 4 — Public Information

Campus Guide — Student Handbook Page

What This Is

A dedicated page for general knowledge about the school — things every student should know but might not. It's NOT a traditional FAQ. It's more like a lightweight "Student Handbook" with categorized accordion sections.

Think of entries like:

  • "Why do we pay ₱50/semester if tuition is free?" → Departmental fee for ID, insurance, and student activities
  • "Where is the registrar's office?" → Building, floor, office hours
  • "How do I join a student organization?" → Process and requirements
  • "What's the academic calendar for this year?" → Key dates overview

Categories to Include

💰 Fees & Financial Info

Departmental fees, misc fees, payment schedules, why certain fees exist despite free tuition

📋 Enrollment & Registration

Step-by-step enrollment process, required documents, deadlines

🏫 Campus Facilities & Services

Library, computer labs, clinic, canteen, study areas, dormitory

🤝 Student Organizations

How to join, list of recognized orgs, what USC does

📅 Academic Calendar

Semester start/end, enrollment periods, exam weeks, holidays

📜 Campus Rules & Regulations

Dress code, ID policy, campus hours, disciplinary procedures

Data Approach — JSON File

Like the events system, store the campus guide data in a JSON file that can be easily edited:
